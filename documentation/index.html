Reton - React Documentation

Introduction

Reton is a React, Next.js and Bootstrap 4 based Multipurpose templates. This has been built with React, ES6+ and as framework Next.js! Reton is a creative responsive template purposely designed and built for Multipurpose Portfolio Landing Page Template.

Setup Development Environment

To kickstart the development of the web app with Reton, at first need to setup the react development environment.

Following tools are needed to setup a react dev environment:

  • Node JS

    • NodeJS works as a node package manager behind the scene. It is recommended to download and install the latest version of Node JS from its official site http://nodejs.org/

Preparation

You'll need to install Node.js >=v12.18+ (Recommended Version) (NPM comes along with it) and serve (a NPM package)

Use command line tool i.e. PowerShell or terminal and navigate to the project(Plaon) root. This is PowerShell in windows and terminal on mac/linux.

Let's have a look on the setup process:

Use terminal and navigate to the project root.

Then run : npm install

Then run : npm run dev

It'll return something like this in the console:

> project@1.0.0 dev drive:\project

> next

[ wait ] starting the development server ...

[ ready ] started server on http://localhost:3000 ...

[ event ] compiled successfully

Now, in the browser go to localhost:3000

For Production Build

Run : npm run build

Then Run : npm run start

See the official next.js doc here

Fonts Used & Main Color Change

By default, the template loads Open+Sans and Poppins font from Google Web Font Services, you can change the font with the one that suits you best. Google Fonts

You will find the font code in the "style.scss file: public/css/style.scss


@import url('https://fonts.googleapis.com/css2?family=Open+Sans:wght@400;600;700&display=swap');
@import url('https://fonts.googleapis.com/css2?family=Poppins:wght@400;500;600;700&display=swap');

$common-font: 'Open Sans', sans-serif;
$heading-font: 'Poppins', sans-serif;

$main-color: #171717;
$paragraph-color: #757575;
$white-color: #fff;
$orange-color: #F68338;
$yellow-color: #FDDB3A;
$blue-color: #5C2CFF;
$pink-color: #ED2C66;
$grey-color: #B7B7B7;
$green-color: #4BFFA5;

Note: After SCSS file run then it will be work
